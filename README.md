# 🤖 AI Interview Simulator

An AI-powered Interview Preparation Platform built with React, Node.js, Express, and Groq AI.

## 🚀 Features

* 🎯 Role-based Interview Questions
* 🤖 AI-Powered Question Generation
* 🧠 Answer Evaluation using LLMs
* 📊 Dynamic Performance Scoring
* 📝 Detailed Feedback for Every Answer
* 🔄 Multi-Step Interview Workflow
* 🎤 AI Voice Interviewer
* 📈 Interview Performance Report
* ⚡ Real-Time Evaluation

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Axios
* CSS3

### Backend

* Node.js
* Express.js
* Groq API

### AI

* Llama 3.3 70B Versatile
* Natural Language Evaluation
* Dynamic Question Generation

## 📸 Workflow

1. Select Interview Role
2. Start AI Interview
3. Answer AI Generated Questions
4. Receive Instant Evaluation
5. Get Score and Feedback
6. View Final Performance Report
